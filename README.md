# Monsoon Atlas

Monsoon Atlas is a four-page responsive website created for a Web Technology activity on website creation. The project is built with plain HTML, CSS, and JavaScript and focuses on slow weekend travel experiences.

## Pages

- `index.html` introduces the concept, layout, and key features.
- `destinations.html` provides filterable route cards and a live weather board.
- `journal.html` adds storytelling and travel notes.
- `contact.html` includes an interactive trip budget planner, smart recommendation, and FAQ section.

## Features

- Responsive layout for desktop and mobile screens
- Semantic HTML structure
- Shared navigation across all pages
- Filter buttons for destination categories
- Live weather board powered by a public forecast API
- Smart route recommendation on the planner page
- Reveal-on-scroll animations
- JavaScript-powered trip budget estimator
- Accordion FAQ interactions

## How to Run

Open `index.html` in any modern browser and navigate using the menu.

## API Used

- Open-Meteo Forecast API for current weather, daily temperature range, rain probability, and wind speed on selected destinations

## Deployment

This project is static, so it is easy to deploy.
